# House Price Prediction

## Project Overview

This project predicts house prices using machine learning techniques. The dataset contains information such as area, number of bedrooms, bathrooms, stories, parking spaces, and other house features.

## Dataset

* Dataset: Housing.csv
* Source: Kaggle Housing Prices Dataset

## Tasks Performed

1. Data Loading and Exploration
2. Data Cleaning and Preprocessing
3. Feature Encoding
4. Model Building

   * Linear Regression
   * Random Forest Regressor
5. Model Evaluation using MAE, RMSE, and R² Score
6. Data Visualization

## Results

* Linear Regression achieved the best performance with an R² score of approximately 0.66.
* Area was found to be the most influential feature affecting house prices.
* Simpler regression models performed better than more complex models for this dataset.
